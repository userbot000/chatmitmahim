# הגדרת תגובה אוטומטית לצ'אטים בפורום

## מה זה עושה?

הסקריפט מתחבר לחשבון הפורום שלך, בודק צ'אטים חדשים שנשלחו אליך, ומחזיר תגובה אוטומטית קבועה לכל צ'אט שעדיין לא קיבל תגובה.

## הגדרת GitHub Actions

### שלב 1: עריכת פרטי ההתחברות

ערוך את הקובץ `auto_reply_chat.py` ושנה את השורות הבאות:

```python
username = "YOUR_USERNAME"  # שם המשתמש שלך בפורום
password = "YOUR_PASSWORD"  # הסיסמה שלך בפורום
auto_reply = "תודה על ההודעה! אני אחזור אליך בהקדם האפשרי."  # התגובה האוטומטית
```

### שלב 2: הפעלת GitHub Actions

1. עבור ללשונית **Actions** בריפוזיטורי
2. אשר את הפעלת Workflows אם מתבקש
3. הסקריפט יתחיל לרוץ אוטומטית פעמיים ביום

## לוח זמנים

הסקריפט רץ אוטומטית:
- **08:00 UTC** (10:00 בישראל)
- **20:00 UTC** (22:00 בישראל)

## בדיקה לפני העלאה ל-GitHub

לפני שמעלים ל-GitHub, מומלץ לבדוק שהכל עובד:

```bash
# הרצת מצב בדיקה - לא שולח הודעות בפועל, רק מראה מה יקרה
python test_auto_reply.py
```

הסקריפט יראה לך:
- ✅ האם ההתחברות הצליחה
- 📋 כמה צ'אטים נמצאו
- 💬 לאילו צ'אטים תישלח תגובה
- ⏭️ אילו צ'אטים ידלגו ולמה

## הרצה ידנית ב-GitHub

אחרי שהעלית את הקבצים ל-GitHub, אתה יכול להריץ את הסקריפט מיד:

1. עבור ללשונית **Actions** בריפוזיטורי
2. בחר ב-workflow **Auto Reply to Chats** מהרשימה בצד שמאל
3. לחץ על הכפתור **Run workflow** (בצד ימין)
4. לחץ על הכפתור הירוק **Run workflow** בחלון שנפתח
5. המתן כ-30 שניות ורענן את הדף
6. לחץ על ההרצה החדשה כדי לראות את הלוגים

## איך זה עובד?

1. הסקריפט מתחבר לחשבון הפורום שלך
2. מקבל רשימה של כל הצ'אטים
3. בודק את ההודעה האחרונה בכל צ'אט
4. אם ההודעה האחרונה לא ממך - שולח תגובה אוטומטית
5. שומר רשימה של צ'אטים שכבר קיבלו תגובה (כדי לא לשלוח פעמיים)

## בדיקת לוגים

לאחר כל הרצה:
1. עבור ללשונית **Actions**
2. לחץ על ההרצה האחרונה
3. לחץ על **auto-reply** לצפייה בלוגים
4. תוכל לראות כמה תגובות נשלחו ולאילו צ'אטים

## הרצה מקומית (לבדיקה)

אם אתה רוצה לבדוק את הסקריפט במחשב שלך:

```bash
# הרצת הסקריפט
python auto_reply_chat.py
```

## התאמה אישית

### שינוי לוח הזמנים

ערוך את הקובץ `.github/workflows/auto_reply.yml`:

```yaml
schedule:
  # דוגמה: ריצה כל 6 שעות
  - cron: '0 */6 * * *'
  
  # דוגמה: ריצה כל יום בשעה 09:00 ו-15:00 UTC
  - cron: '0 9,15 * * *'
```

### שינוי התגובה האוטומטית

ערוך את המשתנה `auto_reply` בקובץ `auto_reply_chat.py`.

## פתרון בעיות

### הסקריפט לא רץ
- ודא ש-GitHub Actions מופעל בריפוזיטורי
- בדוק שערכת את שם המשתמש והסיסמה בקוד

### התחברות נכשלת
- ודא ששם המשתמש והסיסמה נכונים
- בדוק שהחשבון לא נחסם בפורום

### לא נשלחות תגובות
- בדוק את הלוגים ב-Actions
- ודא שיש צ'אטים חדשים שטרם קיבלו תגובה
